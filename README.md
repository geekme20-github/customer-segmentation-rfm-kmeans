# Customer Segmentation: RFM Analysis & K-Means Clustering
## üìå Project Overview.
This project develops an end-to-end customer segmentation engine using transactional retail data. By leveraging RFM (Recency, Frequency, Monetary) analysis and K-Means Clustering, we transform raw transaction logs into actionable business personas.
The goal is to move away from "one-size-fits-all" marketing toward data-driven, personalized customer relationship management.
 
 ## üíº Business Problem.
Treating all customers equally leads to inefficient marketing spend and missed revenue opportunities. This project addresses three core questions:
<br>**Who are our most loyal customers?**
<br>**Which customers are at risk of churning?**
<br>**Which segments have the highest potential for upselling?**

# Data Processing & Cleaning
Using an Online Retail dataset, I performed rigorous cleaning:

<br>**Validation**: Removed missing CustomerID records and cancelled transactions.
<br>**Integrity**: Filtered out non-positive quantities and prices to ensure only valid purchases were modeled.

## 2. Feature Engineering (RFM)
I aggregated transaction-level data into three customer-level metrics:
<br>**Recency ($R$)**: Days since the last purchase.
<br>**Frequency ($F$)**: Total number of unique transactions.
<br>**Monetary ($M$)**: Total revenue generated by the customer.

## 3. Data Transformation
To satisfy the assumptions of the K-Means algorithm:
<br>**Log Transformation**: Applied log1p() to handle highly right-skewed distributions.
<br>**Standardization**: Used StandardScaler to ensure all features exist on the same scale (Mean=0, SD=1).


## 4. Machine Learning Model
<br>**Algorithm**: K-Means Clustering.
<br>**Optimization**: The "Elbow Method" and "Silhouette Analysis" were used to determine the optimal number of clusters ($K=2$).
<br>**Dimensionality Reduction**: PCA (Principal Component Analysis) was utilized to visualize the high-dimensional clusters in a 2D space.

## üìä Cluster Profiles & Insights
<br>Loyal customers have LOW Recency + HIGH Frequency + HIGH Monetary. **Cluster 1**
<br>Non Loyal customers have High Recency + LOW Frequency + Low Monetary value. **Cluster 0**

**Recommendations.**

Create a referral rewards system where they earn ¬£20 credit for each friend they refer. Send special birthday or anniversary offers to strengthen emotional connection.
